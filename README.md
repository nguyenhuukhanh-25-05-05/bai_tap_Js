# Learn JavaScript

A professional development environment for learning JavaScript, built with Vite and Tailwind CSS.

## Features

- **Vite**: Fast HMR (Hot Module Replacement) and optimized builds.
- **Tailwind CSS**: Utility-first CSS framework for modern designs.
- **Modern JavaScript**: Support for ES6+ features.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
