# Premium Glassmorphism Form Project

Welcome to the **Premium Glassmorphism Form** project. This project demonstrates a modern, high-end user interface using vanilla CSS and JavaScript, featuring a sleek "glass" aesthetic with vibrant background animations.

## 🚀 Features

- **Glassmorphism UI**: Beautiful translucent card effect with background blurring.
- **Dynamic Background**: Animated floating bubbles creating a deep, immersive visual experience.
- **Premium Typography**: Integration with Google Fonts ("Outfit") for a professional look.
- **Interactive Elements**:
  - Smooth hover animations on the submit button.
  - Neon-style focus glow on input fields.
  - Professional iconography using Font Awesome.
- **Robust Validation**:
  - Real-time JavaScript validation for required fields, email format, and password length.
  - User-friendly error messages that appear dynamically.

## 🛠️ Technologies Used

- **HTML5**: Structured and semantic markup.
- **CSS3**: Advanced styling including Flexbox, CSS Gradients, `@keyframes` animations, and `backdrop-filter`.
- **JavaScript**: Client-side validation and event handling.
- **Font Awesome**: High-quality vector icons for a better user experience.

## 📂 Project Structure

- `index.html`: The main entry point containing the form structure and validation logic.
- `index.css`: The primary stylesheet for the premium design and animations.
- `test.html`: The destination page after a successful form submission.

## 💻 How to Run

1. Clone this repository to your local machine.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, etc.).
3. Try filling out the form to see the validation and design in action!
