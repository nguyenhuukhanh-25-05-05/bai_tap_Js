# JavaScript Practice Projects

Welcome to my collection of JavaScript practice projects. These projects demonstrate modern UI/UX design and core JavaScript concepts.

## 🛍️ Project 2: Product Explorer (Gallery)
A dynamic e-commerce style product gallery with filtering, searching, and cart functionality.

### Key Features:
- **Shopping Cart**: Real-time cart management with LocalStorage persistence.
- **Product Details Modal**: Immersive modal view for detailed product information.
- **Search & Filter**: Instant search results and category-based filtering.
- **Sorting**: Sort products by price or name (A-Z).
- **Responsive Design**: Optimized for both desktop and mobile devices.

## ✨ Project 1: Premium Glassmorphism Form
A high-end user interface featuring a sleek "glass" aesthetic with vibrant background animations.

### Key Features:
- **Glassmorphism UI**: Beautiful translucent card effect with background blurring.
- **Dynamic Background**: Animated floating bubbles creating a deep, immersive visual experience.
- **Robust Validation**: Real-time JS validation with professional error handling.

## 🛠️ Technologies Used
- **HTML5 & CSS3**: Advanced layouts, animations, and responsive design.
- **Modern JavaScript**: DOM manipulation, array methods, and state management.
- **LocalStorage**: Persistent data storage for the shopping cart.
- **Font Awesome**: Premium vector iconography.

- **HTML5**: Structured and semantic markup.
- **CSS3**: Advanced styling including Flexbox, CSS Gradients, `@keyframes` animations, and `backdrop-filter`.
- **JavaScript**: Client-side validation and event handling.
- **Font Awesome**: High-quality vector icons for a better user experience.

## 📂 Project Structure

- `index.html`: The main entry point containing the form structure and validation logic.
- `index.css`: The primary stylesheet for the premium design and animations.
- `test.html`: The destination page after a successful form submission.

## 💻 How to Run

1. Clone this repository to your local machine.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, etc.).
3. Try filling out the form to see the validation and design in action!
